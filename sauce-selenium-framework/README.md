# Selenium Java Automation – Sauce Demo

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven

## Framework Structure
- BaseTest for driver setup
- Test cases for:
  - Valid Login
  - Invalid Login
  - Locked Out User

## How to Run
1. Clone the repository
2. Import as Maven project
3. Run tests using TestNG
